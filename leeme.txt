Pasos

1.- Descargar archivos
2.- Ingresar a xampp y conectar Apache y Mysql
3.- ingresar a http://localhost/phpmyadmin/
4.- Ejecutar script de creacion de base de datos "database.sql"
5.- Ejecutar scripts de creacion de tablas "regiones.sql", "comunas.sql" y "usuario.sql"
6.- Abrir pagina inicio.php en navegador con ruta localhost/inicio.php/
7.- Ingresar datos requeridos en el formulario
8.- Validar que los datos se registren en la base de datos.
